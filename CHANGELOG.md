## v0.2.0-alpha

### Features
- PGP authentication & message signing
- Server plugins
- Rate limiting plugin
- Nickname validation plugin
- A command-line interface

### Improvements
- CTRL+L shortcut for clearing the buffer
- Better logging
- Message timestamps are now generated by the server, not the client
- `runit` & `systemd` services

### Fixes
- Clients crashing when events were received at a fast pace

## v0.1.0-alpha

This is the very first prerelease of the Bolt protocol, client and server. Do not expect too much from it yet, but a start has been made.
